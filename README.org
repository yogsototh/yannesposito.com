This is the source code of my main blog which is a static website generated from org-mode files.

After multiple attempts, the generator currently use a single =Makefile=
and the org-mode to HTML is handled by using pandoc.

I tried to make my website optimized, so should use very few resources, no js is
needed, no tracker, no comments (on purpose, people can contact me via email if
they wish to).
